# Referências de design — Estimativa PokeFipe

Mockups e artes de referência da função "Quanto pedir por esse Pokémon?"
(botão + modal de estimativa dentro do Avaliar IV do VPLab).

Estes arquivos **não são publicados**: a interface final é construída em
HTML/CSS no próprio design system (responsiva, acessível e leve). O build só
copia `public/`, então nada daqui vai para `dist/`.

| Arquivo | Conteúdo |
| --- | --- |
| `mockup-estimativa-fipe-tema-dourado.png` | Mockup completo do modal, primeira versão (dourada) |
| `mockup-estimativa-fipe-tema-vp-store.png` | Mockup completo do modal no tema VP Store (aprovado) |
| `template-modal-estimativa-fipe.png` | Moldura vazia do modal, sem valores |
| `banner-botao-quanto-pedir-pokemon.png` | Arte do botão com texto "Quanto pedir por esse Pokémon?" |
| `moldura-botao-estimativa-fipe.png` | Moldura vazia do botão (fundo transparente) |
